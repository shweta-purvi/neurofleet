# NeuroFleetX: AI-Driven Urban Mobility Optimization

This project consists of a **React.js** frontend and a **Spring Boot** backend.

## 🚀 Getting Started

### 1. Backend (Spring Boot + MySQL)
The backend provides a RESTful API for authentication, fleet management, and vehicle telemetry.

- **Location:** `./backend`
- **Config:** Update `src/main/resources/application.properties` with your MySQL credentials.
- **Run:**
  ```bash
  cd backend
  mvn spring-boot:run
  ```

### 2. Frontend (React + Vite)
The frontend is a premium React application with real-time analytics and booking interfaces.

- **Location:** `./frontend`
- **Setup:**
  ```bash
  cd frontend
  npm install
  ```
- **Run:**
  ```bash
  npm run dev
  ```

## 🛠 Tech Stack
- **Frontend:** React 19, Vite, Lucide React, Chart.js, Leaflet, Axios, Framer Motion.
- **Backend:** Spring Boot 3, Spring Security (JWT), Spring Data JPA, MySQL.
- **Aesthetics:** High-end glassmorphism design with a dark-mode palette.

## 📁 Project Structure
- `/frontend`: React application (JSX/CSS)
- `/backend`: Spring Boot application (Java)
